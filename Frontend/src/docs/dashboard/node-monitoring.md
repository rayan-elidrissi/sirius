# Monitoring des nœuds

## Tableau de bord

Affichage en temps réel de :
- **Integrity scores** : Score de chaque nœud
- **Réputations** : Score de réputation dynamique
- **Statut** : État de santé (healthy, warning, error)
- **Dernière mise à jour** : Timestamp

## Graphiques

- **Évolution des scores** : Graphique temporel
- **Distribution** : Histogramme des scores
- **Comparaison** : Comparaison entre nœuds

## Alertes

Alertes automatiques pour :
- Score d'intégrité faible
- Divergence détectée
- Nœud non répondant
- Anomalies détectées

## Actions

- **Détails** : Voir l'historique complet d'un nœud
- **Forcer ancrage** : Déclencher un ancrage manuel
- **Exclure** : Exclure un nœud de la surveillance
