# Installation

## Prérequis

- Node.js 18+
- npm ou yarn

## Installation

```bash
git clone <repository>
cd dashboard
npm install
```

## Configuration

Créer un fichier `.env` :

```env
REACT_APP_API_URL=http://localhost:3001
REACT_APP_SUI_RPC_URL=https://fullnode.testnet.sui.io:443
```

## Démarrage

```bash
npm start
```

Le dashboard sera accessible sur http://localhost:3000
