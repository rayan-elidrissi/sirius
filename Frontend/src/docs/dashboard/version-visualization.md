# Visualisation des versions

## Vue DAG

Le dashboard affiche la chaîne des versions sous forme de DAG (Directed Acyclic Graph) :

- **Nœuds** : Représentent les versions
- **Arêtes** : Lien parent → enfant
- **Couleurs** : État de la version (ancrée, en attente, divergente)

## Informations affichées

Pour chaque version :
- Merkle root
- Timestamp
- État d'ancrage sur Sui
- Nombre de blobs
- Signature

## Navigation

- **Zoom** : Pour explorer les détails
- **Filtres** : Par date, état, dataset
- **Recherche** : Par Merkle root ou dataset_id
