# Troubleshooting

## Problèmes courants

### Erreur de connexion au stockage

**Symptôme** : Impossible de se connecter au système de stockage.

**Solution** :
- Vérifier l'URL dans la configuration
- Vérifier les credentials
- Vérifier la connectivité réseau

### Échec d'ancrage sur Sui

**Symptôme** : Les transactions échouent.

**Solution** :
- Vérifier le solde de gas
- Vérifier les permissions du compte
- Vérifier la configuration du réseau Sui

### Divergences non détectées

**Symptôme** : Les divergences ne sont pas détectées.

**Solution** :
- Vérifier la configuration des seuils
- Vérifier que l'AI Layer est actif
- Vérifier les fingerprints générés
