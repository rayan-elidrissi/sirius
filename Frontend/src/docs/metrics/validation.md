# Validation

## Validation des versions

Chaque version est validée :
- Vérification du Merkle root
- Vérification de la signature
- Vérification du lien parent

## Validation des snapshots

Chaque snapshot est validé :
- Vérification des signatures k-of-n
- Vérification de la cohérence
- Vérification de l'ancrage Sui

## Validation des receipts

Chaque receipt est validé :
- Vérification du hash
- Vérification des liens
- Vérification sur Sui

## Tests de régression

Tests automatiques pour :
- Détection de régressions
- Validation des mises à jour
- Assurance qualité
