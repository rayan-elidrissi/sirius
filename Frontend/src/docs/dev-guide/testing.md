# Tests

## Structure des tests

```
tests/
├── unit/
├── integration/
└── e2e/
```

## Exécution

```bash
# Tous les tests
npm test

# Tests unitaires uniquement
npm run test:unit

# Tests d'intégration
npm run test:integration

# Tests e2e
npm run test:e2e
```

## Coverage

```bash
npm run test:coverage
```

## Tests avec Sui

Utiliser le Sui testnet ou un réseau local pour les tests.
