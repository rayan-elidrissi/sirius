# Architecture technique

## Stack technologique

- **Backend** : Node.js, TypeScript
- **Blockchain** : Sui Move
- **Storage** : Ceramic/IPLD
- **Frontend** : React, TypeScript
- **IA** : Modèles de détection de divergences

## Modules principaux

### @sirius/data-layer

Gère le versionnement et les manifests.

### @sirius/ai-layer

Gère la surveillance et la détection.

### @sirius/onchain

Gère l'intégration Sui.

### @sirius/dashboard

Interface utilisateur.
