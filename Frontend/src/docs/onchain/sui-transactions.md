# Transactions Sui

Trois types de transactions sont utilisées pour l'ancrage et l'attestation.

## Types de transactions

- [anchor_epoch](sui-transactions/anchor-epoch.md) - Ancrage des epochs
- [submit_snapshot](sui-transactions/submit-snapshot.md) - Enregistrement des snapshots
- [record_receipt](sui-transactions/record-receipt.md) - Reçus de reproductibilité

## Propriétés communes

- **Public** : Toutes les transactions sont consultables
- **Vérifiable** : Preuves cryptographiques
- **Immuable** : Une fois confirmées, ne peuvent être modifiées
- **Efficaces** : Optimisées pour réduire les coûts
