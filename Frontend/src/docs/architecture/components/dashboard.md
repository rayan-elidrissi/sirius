# Dashboard

Interface web pour visualiser et gérer les datasets et leur traçabilité.

## Fonctionnalités

### Visualisation des versions

- Chaîne des versions (DAG)
- Dernier état ancré sur Sui
- Historique complet

### Monitoring des nœuds

- `integrity_scores` en temps réel
- Réputations des nœuds
- Alertes en cas de divergence

### Reproducibility receipts

- Visualisation des reçus de reproductibilité
- Lien entre modèles et versions de données
- Historique des exécutions IA

## Technologies

- React ou frontend simple
- API backend pour accès aux données
- Visualisations interactives
