# Sécurité et propriétés

## Propriétés de sécurité

### Intégrité

- Merkle roots garantissent l'intégrité des datasets
- Signatures cryptographiques sur chaque version
- Détection automatique des altérations

### Traçabilité

- Historique complet et vérifiable
- Chaîne de versions immuable
- Ancrage sur blockchain pour preuve temporelle

### Disponibilité

- Compatible avec le système de stockage sous-jacent
- Pas de modification du protocole existant

### Reproductibilité

- Receipts cryptographiques liant modèles et données
- Preuves vérifiables sur blockchain

## Modèle de confiance

- **k-of-n signatures** : Consensus distribué
- **Réputation dynamique** : Basée sur l'historique
- **Scoring d'intégrité** : Évaluation continue
